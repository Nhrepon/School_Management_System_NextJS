## NextJs School Management System

## Getting Started

First, run the development server:

```bash
npm run dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Prisma

```bash
npx prisma migrate dev

npx prisma migrate reset

npx prisma db seed

npx prisma studio

```
